# HR Analytics Dashboard (Power BI)

## Project Overview
This project focuses on analyzing employee attrition and workforce trends using Power BI. The dashboard helps HR teams understand key factors influencing employee turnover and performance.

## Objective
Help organizations improve employee performance and retention by identifying patterns in attrition, salary, job roles and demographics.

## Dashboard Preview

![Dashboard](https://github.com/vishalydv-data/HR-Analytics-PowerBI-Dashboard/commit/87ed9d7da9dc1370e17e45e78c68f52e53bcdba4)

## Key Metrics

- Total Employees: 1470
- Average Age: 37
- Average Salary: 6.5K
- Average Working Years: 11
- Performance Rate: 3

## Key Insights

1. Highest attrition observed in the **26–35 age group**
2. Employees with salary **below 5K** show higher attrition
3. **Sales Executives** have the highest attrition among job roles
4. Life Sciences background employees represent the largest workforce share

## Dashboard Features

- Attrition by Age
- Attrition by Education
- Attrition by Salary Slab
- Attrition by Job Role
- Attrition by Years at Company
- Department Filters
